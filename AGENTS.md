# 教育法律文件分析規則

凡涉及案件分析或法律文件撰寫，必須先閱讀：

- `skills/EDUCATION_LEGAL_WRITING.md`
- `skills/EVIDENCE_RULES.md`
- `skills/CITATION_RULES.md`
- `skills/TAIWAN_LEGAL_LANGUAGE.md`
- `skills/REPORT_TEMPLATES.md`
- `skills/REVIEW_CHECKLIST.md`

分析案件前必須：

1. 確認案件資料範圍。
2. 建立來源索引。
3. 檢查頁碼、段落或行號。
4. 區分「當事人主張」、「客觀資料顯示」、「調查認定」及「尚待確認」。
5. 先整理資料，不得直接作成結論。
6. 每項重要敘述均須附來源。
7. 完成後執行引用核對與幻覺檢查。
8. 使用繁體中文及臺灣教育法律用語。
9. 不得虛構事實、證據、頁碼、法規、條號或委員意見。
10. 除非使用者明確要求，不得直接輸出最終認定或懲處建議。

建議工作順序：

文件完整性檢查 → 人物與文件代號 → 程序時間軸 → 事件時間軸 → 各方主張 → 證據對照 → 爭點 → 待補資料 → 文件大綱 → 分章撰寫 → 引用核對。
