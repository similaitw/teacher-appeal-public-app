# 教育法律文件寫作 Skill Pack

## 安裝方式

將本壓縮檔解壓縮後，把以下項目放到 `teacher-appeal-local-ai` 專案根目錄：

- `AGENTS.md`
- `skills/`
- `prompts/`

## Codex 使用方式

在 Codex 輸入：

> 請先閱讀 AGENTS.md 與 skills 目錄中的全部規範，再依 prompts/analyze_case.md 分析 exports 目錄中最新案件。

## ChatGPT 或 Gemini 使用方式

將 `skills/` 中的核心規範上傳，至少包含：

- EDUCATION_LEGAL_WRITING.md
- EVIDENCE_RULES.md
- CITATION_RULES.md
- REPORT_TEMPLATES.md
- REVIEW_CHECKLIST.md

再貼入 `prompts/` 內相對應的提示詞。

## 重要提醒

本 Skill Pack 是文件撰寫與案件整理規範，不取代承辦人、調查委員、申評會、性平會、防制霸凌因應小組、校事會議或法律專業人員的判斷。
