# 教育法律文件審查表

## 事實

- [ ] 是否將單方主張誤寫成事實
- [ ] 是否有無來源的敘述
- [ ] 是否有日期矛盾
- [ ] 是否有人物代號混淆
- [ ] 是否遺漏有利於任一方的資料

## 證據

- [ ] 每項認定是否有證據
- [ ] 是否說明證據相符與矛盾
- [ ] 是否說明不採用證據的理由
- [ ] 是否引用不存在的頁碼
- [ ] 是否以摘要代替原始證據

## 程序

- [ ] 案件類型是否正確
- [ ] 是否使用正確程序
- [ ] 是否確認事件發生時法規
- [ ] 是否確認通知、陳述及救濟程序
- [ ] 是否確認法定期限

## 文字

- [ ] 是否使用繁體中文
- [ ] 是否使用臺灣法律用語
- [ ] 是否過度情緒化
- [ ] 是否出現「惡意」、「說謊」等無證據字眼
- [ ] 是否清楚區分主張與認定

## 結論

- [ ] 結論是否回應所有爭點
- [ ] 結論是否超出調查範圍
- [ ] 是否有證據不足卻強行認定
- [ ] 是否由人工確認最終結論
